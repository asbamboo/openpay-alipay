简介
===================

asbamboo/openpay-alipay 是 `asbamboo/openpay`_ 的一个 `支付渠道扩展模块`_。channel 目录下的代码是对接 `asbamboo/openpay`_ 接口的代码库。

asbamboo/openpay-alipay 也可以作为 `支付宝支付的接口处理库使用`_。alipay-api 目录下的代码是请求支付宝接口的代码库。

web-demo|config|alipay-rsa 是开发过程中调试用的。

.. _asbamboo/openpay: http://www.github.com/asbamboo/openpay
