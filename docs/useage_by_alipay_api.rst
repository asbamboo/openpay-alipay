作为支付宝支付的接口处理库使用
============================================

asbamboo/openpay-alipay 也可以只是作为请求支付宝接口的代码库使用，asbamboo/openpay-alipay 中处理了支付宝对接相关的签名生成与校验。

asbamboo/openpay-alipay 支持如下支付宝接口：

* alipay.trade.app.pay
* alipay.trade.cancel
* alipay.trade.close
* alipay.trade.page.pay
* alipay.trade.precreate
* alipay.trade.query
* alipay.trade.refund
